# CRCHPlaceholder
Displays CustomInfo as a placeholder using [CustomHint](https://github.com/BTF-SCPSL/CustomHint). <br>
Only compatible in Spectator Info <br>
Placeholder: `{custominfo}`

## Configs
Config.yml:
```yaml
c_r_c_h_placeholder:
  is_enabled: true
  debug: false
```
